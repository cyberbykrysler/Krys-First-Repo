## **About This Project**

This website demonstrates:

- **Server setup & domain management**  
  - Hetzner VPS deployment  
  - Domain configuration via Fasthosts  

- **WordPress installation & security hardening**  
  - Nginx web server configuration  
  - Firewall and port management  
  - Fail2Ban for brute-force attack prevention  

- **Command-line & scripting practice**  
  - Ubuntu/Linux commands via PowerShell  
  - Kali Linux VM experimentation  
  - Hands-on security challenges (OverTheWire)  

This project helped me gain practical skills in **server hardening, Linux administration, and cybersecurity fundamentals**, all while creating a live portfolio to showcase my work.
